Install
------------------------------------------------------------------------------

``sqlalchemy_mate`` is released on PyPI, so all you need is:

.. code-block:: console

	$ pip install sqlalchemy_mate

To upgrade to latest version:

.. code-block:: console

	$ pip install --upgrade sqlalchemy_mate
