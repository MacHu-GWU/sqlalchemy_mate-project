Install
------------------------------------------------------------------------------

``sqlalchemy_mate`` is released on PyPI, so all you need is to:

.. code-block:: console

    $ pip install sqlalchemy-mate

To upgrade to latest version:

.. code-block:: console

    $ pip install --upgrade sqlalchemy-mate
